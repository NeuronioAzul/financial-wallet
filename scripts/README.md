# Scripts

Automação e utilitários para desenvolvimento e deploy.

## Scripts Disponíveis

- `inicia-ambiente-dev.sh` - Setup completo do ambiente de desenvolvimento
- `setup.sh` - Configuração inicial do projeto
- `deploy.sh` - Deploy para produção (Hostinger)
- `test.sh` - Executa todos os testes (backend + frontend)

## Uso

```bash
# Setup inicial
./scripts/inicia-ambiente-dev.sh

# Executar testes
./scripts/test.sh
```

Todos os scripts devem ser executados da raiz do projeto.
